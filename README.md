# Owen Rumney Homebrew Tools

This is just a tap for storing some of my tools for easy install with [Homebrew](https://brew.sh/)

## Tools Available

- [Lazytrivy](https:/github.com/owenrumney/lazytrivy) - an interactive Trivy client for scanning and reading Trivy findings
- [Schnappit](https://github.com/owenrumney/schnappit) - a simple MacOS screenshot grabbing tool that does exactly what I need it to do
- [Branch](https://github.com/owenrumney/branch) - a cli for creating conventional commit named branches with optional issue tracking IDs
- [Lazypprof](https://github.com/owenrumney/lazypprof) - an interactive Go Profile tool to analyse profile files or live services

## Installation

```bash
brew tap owenrumney/tools
brew install --cask lazytrivy
brew install --cask schnappit
brew install --cask branch
brew install --cask lazypprof
```
